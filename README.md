# fishbatterywebsite

Static marketing website for the CATCHVOLT portable Li-ion battery positioned for common DAIWA / SHIMANO electric reel applications.

## Pages

- `index.html` - main landing page
- `about.html` - company and product positioning
- `contact.html` - inquiry page
- `wholesale.html` - distributor and OEM / ODM page

## Assets

- `styles.css` - shared site styles
- `favicon.svg` - site icon
- `product-sheet.pdf` - downloadable English product sheet
- `资料/` - product and fishing scene images used by the site

## Local preview

```bash
python3 -m http.server 8000
```

Then open `http://127.0.0.1:8000`.
