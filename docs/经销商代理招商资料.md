# 电动鱼竿锂电池 经销商 / 代理招商资料

## 一、产品简介

这是一款面向海钓、船钓、电动鱼竿/电动轮场景开发的便携式锂电池产品，核心优势在于轻巧便携、供电稳定、电量可视化、配件齐全，适合终端零售、门店展示、套餐销售及品牌定制合作。

与普通通用电池不同，这款产品更适合作为“电动鱼竿便携供电方案”进行销售，场景清晰，目标用户明确，易于通过实拍展示和门店讲解形成成交。

## 二、适合合作的客户类型

- 渔具店
- 船钓团
- 海钓船宿
- 区域代理商
- 渔具品牌方
- 海外经销商
- 私标客户

## 三、产品核心优势

### 1. 场景明确，容易销售

产品直接对应电动鱼竿、电动轮、船钓、海钓用户，不是泛用型模糊产品，销售对象清晰，方便门店快速找到目标客群。

### 2. 差异化明显

相比传统笨重供电设备，这款产品强调：

- 轻便
- 小体积
- 可肩背
- 电量显示
- 船边使用更方便

非常适合通过展示图、视频、试用体验来促成成交。

### 3. 有利于门店展示和视频传播

产品外观完整，配件齐全，且有真实船钓使用画面。对于门店陈列、业务拜访、短视频推广、社群种草都比较友好。

### 4. 适合套餐销售

可与以下产品搭配销售：

- 电动鱼竿
- 电动轮
- 线组
- 收纳配件
- 船钓配件

有利于提升客单价。

### 5. 支持 OEM / ODM

资料显示支持：

- 私标
- 定制颜色
- OEM / ODM 合作
- 小样品起订

适合品牌客户、区域代理和海外批发客户。

- 型号：`RP21-1420`

## 十、招商沟通话术

### 简版话术

这款产品不是普通电池，而是电动鱼竿的便携供电方案。它的好处是场景明确、展示直观、卖点好讲，适合海钓和船钓用户。产品支持私标和颜色定制，也适合门店和代理做差异化经营。

### 面向门店的话术

这类产品在门店里适合做体验式销售，客户一看就能理解用途。再配合船钓场景图和电量显示功能，转化会比单纯讲参数更容易。

### 面向代理的话术

这款产品适合做细分市场切入，因为使用场景很明确。对于想做海钓、电动轮、电动鱼竿相关产品线的代理商来说，容易形成特色品类。

### 面向品牌客户的话术

现有资料显示支持 OEM / ODM、私标和颜色定制，适合品牌客户用更低试错成本测试市场，也适合扩充现有海钓产品线。
