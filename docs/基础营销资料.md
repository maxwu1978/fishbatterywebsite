# 电动鱼竿锂电池 基础营销资料

## 一、产品定位

这是一款面向船钓、海钓、电动鱼竿作业场景的便携式锂电池，可稳定为电动鱼竿/电动轮提供电力，帮助钓手在长时间作钓过程中保持持续供电、降低负担、提升移动灵活性。

建议对外主名称：

- 电动鱼竿便携锂电池
- 船钓电动轮专用锂电池
- 海钓便携供电电池包

## 二、核心参数

- 型号：`RP21-1420`
- 电池能量：`296Wh`
- 额定容量：`20Ah`
- 额定输出：`DC 14.8V`
- 充满电压：`DC 16.8V ± 0.2V`
- 尺寸：约 `220 x 130 x 52 mm`
- 重量：约 `1.64-1.65 kg`
- 防护等级：`IP65`
- 工作温度：`0-60°C`
- 充电温度：`0-55°C`

## 三、核心卖点

### 1. 轻巧便携，适合长时间作钓

机身约 1.65kg，体积小巧，可肩背、可手提，适合船钓过程中频繁移动、变换钓点或长时间站立使用。

### 2. 为电动鱼竿稳定供电

专为电动鱼竿/电动轮使用场景设计，输出稳定，满足船钓、深场钓、长时间作钓的持续供电需求。

常见适配品牌可优先对外写为：

- `DAIWA`
- `SHIMANO`

### 3. 电量可视化，使用更安心

产品自带电量显示面板，可快速掌握剩余电量，减少海上作钓时“怕没电”的焦虑。

### 4. 户外海钓环境更适用

具备 `IP65` 防护能力，可应对海钓、船钓环境中的水花、潮湿、灰尘等常见使用场景。

注意：资料标注说明为“不可浸泡在水中使用或清洗”。

### 5. 多重安全保护，提升使用信心

资料显示产品具备以下安全设计：

- 反接保护
- 过充保护
- 过热断电保护
- 过流/短路/过放保护

适合强调“海上使用更安心”这一购买理由。

### 6. 配件齐全，即买即用

整套包含电池本体、保护套/收纳包、肩背带、输出线组、充电器等，减少用户额外选购成本，也更适合门店和渠道直接销售。

### 7. 支持 OEM / ODM

可提供私标、颜色定制等方案，适合渔具品牌、经销商、船钓团、区域代理合作。
